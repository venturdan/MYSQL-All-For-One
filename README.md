# Project MySQL All For One

O **Project MySQL All For One** foi desenvolvido durante o curso da [Trybe](https://www.betrybe.com/), com o objetivo de praticar a utilização do **MySQL** e consolidar conhecimentos em **banco de dados relacionais**.  
O projeto consiste em uma série de desafios que envolvem a manipulação de dados em tabelas através de comandos SQL, aplicando operações de consulta, filtragem, atualização e agregação.

---

## Objetivo

Aplicar na prática os principais comandos do **SQL**, construindo queries para acessar e manipular informações em um banco de dados.  
O foco do projeto foi desenvolver a habilidade de escrever consultas eficientes e corretas, utilizando **SELECT**, **UPDATE**, **DELETE**, **JOIN**, e funções de agregação.

---

## Funcionalidades

- Criação e manipulação de tabelas de banco de dados  
- Consultas com **SELECT** e **WHERE**  
- Ordenação e filtragem de resultados com **ORDER BY** e **GROUP BY**  
- Atualização e exclusão de registros com **UPDATE** e **DELETE**  
- Uso de **JOINs** para combinar dados de múltiplas tabelas  
- Aplicação de funções de agregação como **COUNT**, **AVG**, **SUM**, **MAX** e **MIN**  

---

## Tecnologias Utilizadas

- **MySQL**  
- **Docker**  
- **Workbench / CLI MySQL**  
- **JavaScript (para execução de testes automatizados)**  

---

## Conceitos Praticados

- Escrita e execução de **queries SQL**  
- Manipulação de **dados relacionais**  
- Aplicação de **filtros e condições lógicas**  
- Uso de **funções de agregação e agrupamento**  
- Compreensão de **relacionamentos entre tabelas (JOINs)**  
- Boas práticas de **organização e legibilidade de consultas**
